# simulador-colisoes-fisica
Simulador de colisões com múltiplas entidades.

1. Pré-Requisitos e Instalação
Para executar a simulação, você precisará do Python 3.12.0 e da biblioteca VPython.
1.1
Instalação do VPython: pip install vpython

2. Configurações
Todos os parâmetros da simulação (número de esferas, dimensões, velocidades, etc.) são lidos do arquivo config.json.
